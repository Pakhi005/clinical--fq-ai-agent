# Clinical FAQ AI Agent with RAG Pipeline — Complete Build Guide
**Build Time: 2 Days | Tech: Python, LangChain, Gemini API, FAISS, FastAPI, React.js**

## 📋 PROJECT OVERVIEW

**What you'll build:**
A web app where users ask questions about clinical research. The app retrieves relevant information from a document knowledge base and answers using an LLM.

**How it works:**
1. Load clinical PDFs → chunk text → create embeddings
2. Store embeddings in FAISS vector database
3. User asks a question → retrieve relevant documents → send to Gemini API
4. Gemini generates grounded answer based on context
5. Answer displayed on React frontend with source documents

**Skills demonstrated:**
- RAG architecture ✅
- LangChain ✅
- Vector databases (FAISS) ✅
- LLM integration (Gemini) ✅
- REST API design ✅
- Frontend integration ✅
- Guardrails (safety) ✅

### Step 6: Create RAG Pipeline Code

Create `rag_pipeline.py` in backend:

```python
# rag_pipeline.py
import os
from dotenv import load_dotenv
from langchain.embeddings import HuggingFaceEmbeddings
from langchain.vectorstores import FAISS
from langchain.document_loaders import PyPDFLoader
from langchain.text_splitter import CharacterTextSplitter
from PyPDF2 import PdfReader
import json

load_dotenv()

class RAGPipeline:
    def __init__(self, index_path="faiss_index"):
        self.index_path = index_path
        self.embeddings = HuggingFaceEmbeddings(
            model_name="sentence-transformers/all-MiniLM-L6-v2"
        )
        self.vectorstore = None
        self.documents = []
        
    def load_documents_from_text(self, documents_list):
        """Load documents from list of dicts (title + content)"""
        from langchain.schema import Document
        
        loaded_docs = []
        for doc in documents_list:
            doc_obj = Document(
                page_content=doc["content"],
                metadata={"source": doc["title"]}
            )
            loaded_docs.append(doc_obj)
        
        self.documents = loaded_docs
        print(f"✅ Loaded {len(loaded_docs)} documents")
        return loaded_docs
    
    def load_documents_from_pdf(self, pdf_folder):
        """Load documents from PDF files"""
        from langchain.schema import Document
        
        docs = []
        for filename in os.listdir(pdf_folder):
            if filename.endswith(".pdf"):
                pdf_path = os.path.join(pdf_folder, filename)
                print(f"Loading {filename}...")
                
                reader = PdfReader(pdf_path)
                for page_num, page in enumerate(reader.pages):
                    text = page.extract_text()
                    doc = Document(
                        page_content=text,
                        metadata={"source": f"{filename} (page {page_num+1})"}
                    )
                    docs.append(doc)
        
        self.documents = docs
        print(f"✅ Loaded {len(docs)} pages from PDFs")
        return docs
    
    def chunk_documents(self, chunk_size=500, chunk_overlap=100):
        """Split documents into chunks"""
        splitter = CharacterTextSplitter(
            chunk_size=chunk_size,
            chunk_overlap=chunk_overlap
        )
        
        chunks = splitter.split_documents(self.documents)
        print(f"✅ Split into {len(chunks)} chunks")
        return chunks
    
    def create_vector_store(self):
        """Create FAISS vector store"""
        if not self.documents:
            raise ValueError("No documents loaded. Load documents first.")
        
        chunks = self.chunk_documents()
        
        # Create FAISS index
        self.vectorstore = FAISS.from_documents(
            chunks, 
            self.embeddings
        )
        
        # Save to disk
        self.vectorstore.save_local(self.index_path)
        print(f"✅ Vector store created and saved to {self.index_path}")
        
        return self.vectorstore
    
    def load_vector_store(self):
        """Load existing FAISS index"""
        if os.path.exists(self.index_path):
            self.vectorstore = FAISS.load_local(
                self.index_path, 
                self.embeddings
            )
            print(f"✅ Vector store loaded from {self.index_path}")
            return self.vectorstore
        else:
            raise FileNotFoundError(f"Vector store not found at {self.index_path}")
    
    def retrieve_context(self, query, k=3):
        """Retrieve top k relevant documents for a query"""
        if self.vectorstore is None:
            self.load_vector_store()
        
        results = self.vectorstore.similarity_search_with_scores(query, k=k)
        
        documents = []
        for doc, score in results:
            documents.append({
                "content": doc.page_content,
                "source": doc.metadata.get("source", "Unknown"),
                "score": float(score)
            })
        
        return documents


# Initialize pipeline
rag = RAGPipeline()

if __name__ == "__main__":
    # Test: Load and create vector store
    from sample_documents import CLINICAL_DOCS
    
    rag.load_documents_from_text(CLINICAL_DOCS)

```

### Step 7: Create LLM Integration Code

Create `llm_service.py` in backend:

```python
# llm_service.py
import os
from dotenv import load_dotenv
from langchain_google_genai import ChatGoogleGenerativeAI
from langchain.prompts import PromptTemplate
from rag_pipeline import rag

load_dotenv()

class LLMService:
    def __init__(self):
        self.llm = ChatGoogleGenerativeAI(
            model="gemini-pro",
            google_api_key=os.getenv("GOOGLE_API_KEY"),
            temperature=0.3
        )
        
        self.prompt_template = PromptTemplate(
            input_variables=["context", "question"],
            template="""You are a helpful assistant for clinical research questions.
            
Use the following context to answer the user's question. If the answer is not in the context, 
say "I don't have enough information to answer that question." Do NOT make up information.

Context:
{context}

Question: {question}

Answer:"""
        )
    
    def generate_answer(self, question: str, context_documents: list) -> dict:
        """Generate answer based on retrieved documents"""
        
        # Combine context
        context_text = "\n\n".join([
            f"[{doc['source']}]\n{doc['content']}"
            for doc in context_documents
        ])
        
        # Create prompt
        prompt = self.prompt_template.format(
            context=context_text,
            question=question
        )
        
        # Get answer from Gemini
        response = self.llm.invoke(prompt)
        answer = response.content
        
        # Add guardrail check
        if self.is_safe_answer(answer, question):
            return {
                "answer": answer,
                "sources": [doc["source"] for doc in context_documents],
                "safe": True
            }
        else:
            return {
                "answer": "I cannot provide a reliable answer to this question.",
                "sources": [],
                "safe": False
            }
    
    def is_safe_answer(self, answer: str, question: str) -> bool:
        """Check if answer is safe and relevant"""
        
        # If it says "don't know", it's safe
        if "don't have" in answer.lower() or "insufficient" in answer.lower():
            return True
        
        # If answer is reasonable length, it's likely good
        if 50 < len(answer) < 2000:
            return True
        
        return False


# Initialize service
llm_service = LLMService()

if __name__ == "__main__":
    # Test
    question = "What is a Phase II clinical trial?"
    context = rag.retrieve_context(question)
    
    result = llm_service.generate_answer(question, context)
    print(f"\nQuestion: {question}")
    print(f"Answer: {result['answer']}")
    print(f"Sources: {result['sources']}")
```

```python
# main.py
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from pydantic import BaseModel
from rag_pipeline import rag
from llm_service import llm_service
from dotenv import load_dotenv

load_dotenv()

app = FastAPI(title="Clinical FAQ AI Agent")

# Enable CORS for React frontend
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# Initialize RAG on startup
@app.on_event("startup")
async def startup_event():
    print("Loading RAG pipeline...")
    try:
        rag.load_vector_store()
        print("✅ RAG pipeline loaded")
    except FileNotFoundError:
        print("⚠️ Vector store not found. Creating new one...")
        from sample_documents import CLINICAL_DOCS
        rag.load_documents_from_text(CLINICAL_DOCS)
        rag.create_vector_store()
        print("✅ Vector store created")

# Request model
class ChatRequest(BaseModel):
    question: str

class ChatResponse(BaseModel):
    answer: str
    sources: list
    safe: bool

# API endpoint
@app.post("/chat", response_model=ChatResponse)
async def chat(request: ChatRequest):
    """Chat endpoint for clinical FAQ"""
    
    question = request.question
    
    # Retrieve relevant documents
    context = rag.retrieve_context(question, k=3)
    
    # Generate answer using LLM
    result = llm_service.generate_answer(question, context)
    
    return ChatResponse(
        answer=result["answer"],
        sources=result["sources"],
        safe=result["safe"]
    )

@app.get("/health")
async def health():
    """Health check"""
    return {"status": "ok"}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

```javascript
// src/App.js
import React, { useState, useRef, useEffect } from 'react';
import axios from 'axios';
import './App.css';

function App() {
  const [messages, setMessages] = useState([]);
  const [input, setInput] = useState('');
  const [loading, setLoading] = useState(false);
  const messagesEndRef = useRef(null);

  const scrollToBottom = () => {
    messagesEndRef.current?.scrollIntoView({ behavior: 'smooth' });
  };

  useEffect(() => {
    scrollToBottom();
  }, [messages]);

  const handleSend = async (e) => {
    e.preventDefault();
    
    if (!input.trim()) return;

    // Add user message
    const userMessage = { role: 'user', content: input };
    setMessages([...messages, userMessage]);
    setInput('');
    setLoading(true);

    try {
      const response = await axios.post(
        'http://localhost:8000/chat',
        { question: input }
      );

      const assistantMessage = {
        role: 'assistant',
        content: response.data.answer,
        sources: response.data.sources
      };

      setMessages(prev => [...prev, assistantMessage]);
    } catch (error) {
      const errorMessage = {
        role: 'assistant',
        content: 'Sorry, there was an error processing your question.'
      };
      setMessages(prev => [...prev, errorMessage]);
      console.error('Error:', error);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="App">
      <div className="container">
        <div className="header">
          <h1>Clinical Research FAQ Assistant</h1>
          <p>Ask questions about clinical trials and research</p>
        </div>

        <div className="chat-box">
          {messages.length === 0 ? (
            <div className="welcome">
              <h2>Welcome to Clinical FAQ AI Agent</h2>
              <p>Ask me anything about clinical trials, eligibility, safety, and more.</p>
              <div className="examples">
                <p>Try asking:</p>
                <ul>
                  <li>"What are the types of clinical trials?"</li>
                  <li>"What is informed consent?"</li>
                  <li>"What happens during screening?"</li>
                </ul>
              </div>
            </div>
          ) : (
            messages.map((msg, idx) => (
              <div key={idx} className={`message ${msg.role}`}>
                <div className="message-content">{msg.content}</div>
                {msg.sources && msg.sources.length > 0 && (
                  <div className="sources">
                    <strong>Sources:</strong>
                    <ul>
                      {msg.sources.map((source, i) => (
                        <li key={i}>{source}</li>
                      ))}
                    </ul>
                  </div>
                )}
              </div>
            ))
          )}
          {loading && (
            <div className="message assistant">
              <div className="loading">
                <span></span><span></span><span></span>
              </div>
            </div>
          )}
          <div ref={messagesEndRef} />
        </div>

        <form className="input-form" onSubmit={handleSend}>
          <input
            type="text"
            value={input}
            onChange={(e) => setInput(e.target.value)}
            placeholder="Ask a clinical research question..."
            disabled={loading}
          />
          <button type="submit" disabled={loading}>
            {loading ? 'Sending...' : 'Send'}
          </button>
        </form>
      </div>
    </div>
  );
}

export default App;
```

2. **Embedding**: Generate embeddings using Sentence Transformers
3. **Retrieval**: User query → retrieve top 3 relevant documents from FAISS
4. **Generation**: Send context + query to Gemini API → get answer
5. **Safety**: Check response for hallucinations → return safe answer

## 📊 WHAT TO TELL IN INTERVIEWS

**"What does this project demonstrate?"**
> I built an end-to-end RAG system that answers clinical research questions. It loads PDF documents, creates embeddings, stores them in a FAISS vector database, retrieves relevant documents for a user query, and uses Gemini API to generate grounded answers. I added guardrails to prevent hallucinations.

**"Why RAG instead of just LLM?"**
> RAG ensures answers are grounded in actual documents and reduces hallucinations. For clinical information, accuracy is critical — users need sourced, reliable information.

**"How do you prevent bad answers?"**
> I implemented a safety check that verifies answers are within expected parameters. If the LLM can't find relevant information, it says "I don't know" instead of making something up.

**"How would you improve this?"**
> Add PDF upload, multi-turn conversations, better guardrails for medical misinformation, user feedback loop to improve retrieval quality.
